# Цель:
Требуется разработать программу, которая выполняет HTTP-запросы к сервису _https://tools-httpstatus.pickup-services.com/_ и обрабатывает ответы следующим образом:

- При получении статус-кодов 1xx, 2xx, 3xx- логировать содержимое ответа (как минимум статус-код и тело ответа)
- При получении статус-кодов 4xx и 5xx - генерировать исключительную ситуацию (exception).
- При вызове скрипта, он должен выполнить 5 разных запросов и обработать их соответствующим образом.
- Результат скрипта необходимо логировать в консоль.

# Требования:

- Для выполнения данного задания, вы можете использовать python, либо реализовать скрипт на bash.
- Скрипт должен быть универсальным и легко читаемым, чтобы его можно было использовать как основу для дальнейших шагов.
